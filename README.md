# CarInsurance

ASP.NET MVC project for car insurance quote calculation.

## How to run

1. open the project in visual studio or vs code
2. run dotnet run
3. go to /Insuree/Index to see the list
4. go to /Insuree/Admin to see the admin page

## Features

- add new insuree and get a quote automatically
- edit, delete, view details for each insuree
- admin page show all quotes with name and email
